# 🚀 Prologue – AI Learning Platform Landing Page

A modern and responsive **AI-powered Learning Platform Landing Page** built using **Next.js**, **React**, **Tailwind CSS**, and **Framer Motion**. The website showcases the concept of an interactive visual learning platform through a clean interface, smooth animations, and responsive design.

> **Note:** This project is currently a **static frontend landing page** with no backend or AI integration.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/prologue.git
```

Go to the project directory

```bash
cd prologue
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser

```text
http://localhost:3000
```
